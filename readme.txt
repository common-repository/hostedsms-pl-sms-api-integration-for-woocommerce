HostedSMS.pl - SMS API integration for WooCommerce

== Description ==

Wtyczka służy do wysyłania powiadomień SMS o zmianach statusów do klientów sklepu opartego o WooCommerce dla WordPress. Możliwe jest skonfigurowanie powiadomień o zmianach dla 4 rodzajów statusów sklepu WooCommerce.

= EN =

The plugin is used to send SMS notifications about status changes to customers of a WooCommerce-based store for WordPress. It is possible to configure notifications for changes in 4 types of WooCommerce store statuses.

== Installation	 ==

Instalacja wtyczki możliwa jest z poziomu zaplecza administracyjnego WordPress lub wgrywając zawartość paczki do katalogu wp-content/plugins instalacji WordPress. Po wgraniu wtyczki należy dokonać jej aktywacji z zaplecza administratora WordPress. W następnym kroku wymagane jest podanie danych do logowania serwisu HostedSMS, konfiguracja obsługiwanych statusów i treści wysyłanych wiadomości SMS.

= EN =

The plugin can be installed either through the WordPress administrative backend or by uploading the contents of the package to the wp-content/plugins directory of your WordPress installation. After uploading the plugin, it needs to be activated from the WordPress admin backend. The next step involves providing login credentials for the HostedSMS service, configuring supported statuses, and the content of the sent SMS messages.
